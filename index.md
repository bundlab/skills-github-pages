## 🛠️ Quick Reference Command Cheat Sheet

| Command | What it does |
| :--- | :--- |
| `git init` | Initializes a brand new local repository |
| `git status` | Shows modified files and staged changes |
| `git add .` | Stages all changed files for the next commit |
| `git commit -m "message"` | Saves your staged snapshot permanently |
| `git push origin main` | Uploads local commits to GitHub |

> 💡 **Tip for Students:** Practice makes perfect. Try opening your terminal alongside these lessons and running the commands on a test repository!
